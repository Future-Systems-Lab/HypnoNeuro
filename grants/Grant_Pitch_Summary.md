# HypnoNeuro – Grant Pitch Summary

HypnoNeuro is a decentralized mental wellness platform using NFTs and crypto incentives to reward therapeutic healing. This project targets mental health disparities by offering anonymous, gamified therapy access across mobile and Web3 environments.

**Key Features:**
- Utility token (HNT) powers rewards and session payments
- NFT tiers tied to therapy progress (L1–L3)
- HIPAA-aligned infrastructure using decentralized IDs and encryption
- Peer support, trauma recovery rooms, and practitioner onboarding

**Current Status:**
- Token contracts deployed and verified (HNT on Sepolia)
- App under active development (Play Store approved)
- White paper complete, grant applications initiated
